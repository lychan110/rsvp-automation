\# Brand and Visual Rules



Read this file when styling slides, choosing diagram colors, or applying Siemens brand conventions.



\## General Rules



1\. Use the Siemens palette consistently.

2\. Keep strong headline hierarchy.

3\. Keep clutter low and contrast clear.

4\. Use dark-first appearance when appropriate.



\## Semantic Color Mapping



| Color | Use For |

|---|---|

| Navy | Decisions / control points |

| Teal / petrol | Approvals / highlights |

| Mid-blue | Section dividers |

| Light gold | Optional / advisory / future |

| Light blue / green | Core and validated states |
