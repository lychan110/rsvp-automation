# Siemens Presentation Brand Cheatsheet

## Purpose
Use this as the first-reference brand guide for technical storytelling decks.

## Core Principles
- Clarity first: keep composition simple and avoid clutter.
- Purposeful layout: only include elements that support the message.
- Impact with restraint: strong headline + one primary visual per slide.
- Dark-first brand impression is preferred when suitable.

## Color Quick Reference (Screen RGB)
- Siemens Petrol: `#009999`
- Light Petrol: `#00C1B6`
- Deep Blue: `#000028`
- Bold Green: `#00FFB9`
- Bold Blue: `#00E6DC`
- Light Sand: `#F3F3F0`
- White: `#FFFFFF`

## Recommended Deck Palette Mapping
- Primary dark background: Deep Blue family (`#1B2A4A` / `#000028`)
- Accent/approval highlight: Teal/Petrol (`#009FC8` / `#009999`)
- Optional/advisory highlight: light gold (`#FFF3CD`)
- Core/system panels: light blue (`#EAF4FB`)
- Validated/success panels: light green (`#EAFAF1`)

## Layout Rules
- Keep one headline message per slide.
- Avoid dense blocks of text; use bullets.
- Main slides: simple visuals (3-5 key elements).
- Backup slides: detailed technical content.
- If whitespace is intentional, center content.

## Figure Rules
- Full-width figures should be 16:9 and placed below title.
- Wide figures: place text above/below.
- Tall figures: place text left/right.
- Resize figure internals (text, line width, node size) to remain readable.

## Storytelling Arc
1. Background and motivation
2. Why it matters (team/stakeholders/company)
3. Methods
4. Results/evidence
5. Conclusion tied to benefits
6. Future outlook
