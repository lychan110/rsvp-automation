# Data Directory

This folder contains sample invitee data and example exports used by the automated RSVP workflow.

Files:
- `sample_invitees.csv` — example invitee list with required columns.
- `sample_invitees_sheet.json` — JSON representation of the sheet structure.
- `form_responses.csv` — sample Google Form response export for RSVP sync.

Required columns for CSV inputs:
- `FirstName`
- `LastName`
- `Email`
- `Mailing Address`
- `Number of Attendees`
- `InviteSent`
- `RSVP_Status`
- `RSVP_Timestamp`
